# API接口文档

## 概述

本文档介绍了 RMG 系统的 API 接口文档，包括接口使用说明、请求参数、响应参数、错误码等内容。

## 接口列表

### 1-登录接口

#### 请求URL

- URL：`/api/login`
- Method：`POST`

#### 请求参数

列出接口的请求参数，包括参数名称、类型、是否必填、默认值、描述等。

| 参数名称 | 类型   | 是否必填 | 描述   |
| -------- | ------ | -------- | ------ |
| username | String | 必填     | 用户名 |
| password | String | 必填     | 密码   |

#### 响应参数

列出接口的响应参数，包括参数名称、类型、描述等。

| 参数名称 | 类型   | 描述                                          |
| -------- | ------ | --------------------------------------------- |
| err      | Number | 0表示用户不存在，1表示登录失败，2表示登录成功 |

### 2-注册接口

#### 请求URL

- URL：`/api/register`
- Method：`POST`

#### 请求参数

列出接口的请求参数，包括参数名称、类型、是否必填、默认值、描述等。

| 参数名称 | 类型   | 是否必填 | 描述   |
| -------- | ------ | -------- | ------ |
| username | String | 必填     | 用户名 |
| nickname | String | 必填     | 昵称   |
| password | String | 必填     | 密码   |

#### 响应参数

列出接口的响应参数，包括参数名称、类型、描述等。

| 参数名称 | 类型   | 描述                                                      |
| -------- | ------ | --------------------------------------------------------- |
| err      | Number | 0表示用户已存在，请直接登录；1表示注册成功；2表示注册失败 |

### 3-获取所有积分接口

#### 请求URL

- URL：`/api/credits`
- Method：`GET`

#### 请求参数

无

#### 响应参数

响应参数为一数组，包含每个用户的用户名（username），昵称（nickname），积分（credit）
